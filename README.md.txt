# RedSea250

Sistema IoT para controle e monitoramento de iluminação e temperatura para aquário marinho utilizando ESP32, Spring Boot e React.

---

## 🏗 Arquitetura

ESP32 → Spring Boot API (JWT) → PostgreSQL  
Frontend React → Consome API protegida

---

## 🚀 Tecnologias

### Backend
- Java 17
- Spring Boot 3
- Spring Security
- JWT
- PostgreSQL

### Frontend
- React
- Vite
- TailwindCSS

---

## 🔐 Segurança

- Autenticação via JWT
- Rotas protegidas
- Filtro customizado de validação de token
